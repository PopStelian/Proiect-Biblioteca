# 📚 Biblioteca Online - Full Stack Application

Aceasta este o aplicație completă pentru gestionarea unei biblioteci, care permite vizualizarea cărților, gestionarea membrilor și efectuarea împrumuturilor în timp real.

## 🚀 Demo Live
- **Frontend (Vercel): https://proiect-biblioteca.onrender.com/
- **Backend (Render): https://proiect-biblioteca.vercel.app/

## 🛠️ Tehnologii Utilizate

### Frontend
* **React.js** (Vite) - Interfață dinamică și responsivă.
* **Axios** - Pentru comunicarea cu API-ul.
* **React Router** - Navigare între pagini.

### Backend
* **Java 21** cu **Spring Boot 3**.
* **Spring Data JPA** - Pentru interacțiunea cu baza de date.
* **PostgreSQL** - Baza de date relațională.

### Cloud & Deployment
* **Aiven** - Găzduire bază de date PostgreSQL.
* **Render** - Deployment Backend (Dockerizat).
* **Vercel** - Deployment Frontend.


## 📋 Funcționalități
- [x] Vizualizarea catalogului de cărți.
- [x] Căutare dinamică după titlu sau autor.
- [x] Sistem de autentificare (Login/Register).
- [x] Gestionarea împrumuturilor (scăderea automată a stocului).
- [x] Conexiune live cu baza de date din Cloud.
